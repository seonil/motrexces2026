**제품 1 - DashCam:**
- DashCam
- Advanced dual-channel recording system for comprehensive vehicle monitoring.
- Recording: Dual-Channel (Front + Rear)
- Coverage: Driving / Parking / Incident Monitoring
- Hardware: New-Generation Main PCB & Rear Camera Ass'y
- Performance: Stable, High-Quality Video
- Connectivity: App-Integrated Control

**제품 2 - AC Home EV Charger:**
- AC Home EV Charger
- Our charging solution redefines how energy is connected to life. Portable chargers provide convenience on the go, while home chargers work efficiently to build a smart and flexible EV ecosystem.
- Power Options: 7kW / 12kW / 22kW
- Compatibility: Type 1, Type 2, NACS
- Connectivity: Mobile App, RFID, Cloud
- Features: Load Balancing, IP54, 5m Cable
- Setup: Single & 3-Phase Support

**제품 3 - Portable Charger:**
- Portable Charger
- Lightweight portable charging hardware enabling flexible deployments for fleets, pop-up events, and emergency backup power.
- Power Options: 3.5kW / 2.9kW / 2.2kW
- Voltage: Single Phase, 183–277Vac
- Connectors: Type 1 / Type 2 / NACS
- Standard: IEC 62752
